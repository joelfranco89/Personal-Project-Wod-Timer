ABOUT

Wod Timer - Simple was specifically designed for functional fitness enthusiasts that need a clear timer to work out while being able to setup timers for TIME, EMOM, and AMRAP workouts. Although the timer was designed for fitness athletes, all three timers can be used by anyone for any purpose. The three timers include;

TIME - set the timer to count up and include a time cap (maximum amount of time needed) for the specific workout.

EMOM - set the total number of rounds you would like and the time length of the countdown. Each round will equal to 1 minute

AMRAP - set the start time of the workout and the timer will count down.

TABATA - This timer is automatically set up to run the standard tabata workout. There will be 8 total rounds with 20 seconds of work and 10 seconds of rest.

We have included a Delayed Start feature which allows you to set a timer before the actual countdown timer starts. This will help when trying to start two seperate timers at the same time. This is an optional feature so you do not need to use it.

This is a community friendly timer. We encourage you to send us a message if you feel there should be a feature that could really make your experience better. We want to make this the most effective timer while also making it user friendly without all the extra unnecessary features. That's why we want to be effective, but SIMPLE!
